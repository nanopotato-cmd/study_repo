## Основные команды Git

Теперь посмотрим на ключевые команды Git:

```
**Команда**                  **Что делает**
git init                     Создаёт новый пустой репозиторий Git
git clone <url>              Клонирует репозиторий с сервера и копирует всю его историю
git status                   Показывает, какие файлы изменены и что находится в staging
git add <файл>               Добавляет изменения в staging area
git commit -m ""             Создаёт новый коммит из staging area
git diff                     Показывает разницу между текущими и закоммиченными файлами
git log                      Показывает историю коммитов
git checkout <ветка>         Переключает контекст HEAD на другую ветку
git reset / revert           Используются для откатов изменений (разные подходы) 
```
