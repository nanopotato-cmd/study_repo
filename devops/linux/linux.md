## 1. Основы Linux

### Что такое Linux?
Linux — семейство Unix-подобных операционных систем с открытым исходным кодом.

### Структура файловой системы

| Директория | Назначение |
|------------|------------|
| `/` | Корневая директория |
| `/home` | Домашние папки пользователей |
| `/etc` | Конфигурационные файлы |
| `/var` | Переменные данные (логи, кэш) |
| `/usr` | Пользовательские программы и утилиты |
| `/bin` | Основные системные команды |
| `/tmp` | Временные файлы |
| `/opt` | Дополнительное ПО |

## 2. Навигация по файловой системе

```bash
pwd                    # Показать текущую директорию
ls                     # Список файлов в папке
ls -la                 # Подробный список (включая скрытые)
ls -lh                 # Человеко-читаемые размеры файлов

cd /путь/к/папке       # Перейти в папку
cd ~                   # Перейти в домашнюю папку
cd ..                  # Перейти на уровень выше
cd -                   # Вернуться в предыдущую папку

tree                   # Показать дерево папок (требуется установка)
```

## 3. Работа с файлами и папками

```bash
# Создание
touch file.txt         # Создать пустой файл
mkdir folder           # Создать папку
mkdir -p path/to/folder  # Создать вложенные папки

# Копирование
cp source.txt dest.txt     # Копировать файл
cp -r source_dir dest_dir  # Копировать папку рекурсивно

# Перемещение/переименование
mv old.txt new.txt          # Переименовать файл
mv file.txt /path/to/dir/   # Переместить файл

# Удаление
rm file.txt              # Удалить файл
rm -r folder             # Удалить папку с содержимым
rm -f file.txt           # Принудительное удаление (без запроса)
rm -rf folder            # Удалить папку принудительно (ОСТОРОЖНО!)

# Ссылки (ярлыки)
ln -s /путь/к/цели link_name  # Создать символическую ссылку

```

```bash
wc -w file.txt           # Количество слов
```

## 5. Права доступа

### Что такое права доступа?

В Linux каждый файл и папка имеют три категории пользователей и три типа прав. Это основа безопасности системы.

### Три категории пользователей

| Категория | Обозначение | Кто входит |
|-----------|-------------|------------|
| **Владелец** | `u` (user) | Пользователь, который создал файл |
| **Группа** | `g` (group) | Пользователи, входящие в группу файла |
| **Остальные** | `o` (others) | Все остальные пользователи системы |

### Три типа прав

| Право | Обозначение | Для файла | Для папки |
|-------|-------------|-----------|-----------|
| **Чтение** | `r` (read) | Просмотр содержимого | Просмотр списка файлов (`ls`) |
| **Запись** | `w` (write) | Изменение содержимого | Создание/удаление файлов внутри |
| **Исполнение** | `x` (execute) | Запуск как программы | Вход в папку (`cd`) |

---

### Формат отображения прав

Команда `ls -la` показывает права в таком формате:

```bash
-rwxr-xr-- 1 user group 1234 Mar 29 10:00 script.sh
d rwx r-x r-- 1 user group 4096 Mar 29 10:00 my_folder
│ │   │   │
│ │   │   └─── права для остальных (others)
│ │   └─────── права для группы (group)
│ └─────────── права для владельца (user)
└───────────── тип файла (- обычный, d папка, l ссылка)

```

## 7. Пользователи и группы

```bash
whoami                   # Текущий пользователь
id                       # Информация о пользователе
users                    # Список вошедших пользователей
sudo command             # Выполнить команду от root

# Управление пользователями (требуется sudo)
sudo useradd username    # Создать пользователя
sudo passwd username     # Установить пароль
sudo userdel username    # Удалить пользователя
sudo usermod -aG group username  # Добавить пользователя в группу

su - username            # Переключиться на другого пользователя
```

## 8. Поиск файлов и текста

```bash
# Поиск файлов
find / -name "file.txt"           # Искать по имени
find . -type f -name "*.md"       # Искать .md файлы в текущей папке
find /home -type d -name "devops" # Искать папки по имени

# Поиск текста в файлах
grep "text" file.txt              # Найти строки с текстом
grep -r "pattern" /path/          # Рекурсивный поиск
grep -i "text" file.txt           # Без учета регистра
grep -n "text" file.txt           # Показать номера строк
grep -v "text" file.txt           # Исключить строки с текстом

# Комбинация find + grep
find . -name "*.py" -exec grep -l "import" {} \;
```

## 9. Работа с архивами

```bash
# tar (самый распространенный)
tar -czf archive.tar.gz folder/   # Создать архив (.tar.gz)
tar -xzf archive.tar.gz           # Распаковать архив
tar -tf archive.tar.gz            # Посмотреть содержимое

# zip/unzip
zip -r archive.zip folder/        # Создать zip архив
unzip archive.zip                  # Распаковать zip
unzip archive.zip -d target_dir    # Распаковать в папку

# Другие форматы
gzip file.txt          # Сжать (.gz)
gunzip file.txt.gz     # Распаковать
```

## 13. Перенаправление и конвейеры (пайпы)

```bash
# Перенаправление вывода
command > file.txt       # Записать вывод в файл (перезаписать)
command >> file.txt      # Добавить вывод в конец файла
command 2> error.log     # Перенаправить ошибки
command &> output.txt    # Перенаправить всё (stdout + stderr)

# Конвейеры (передача вывода одной команды на вход другой)
command1 | command2      # Передать вывод command1 в command2
ls -la | grep "txt"      # Найти txt файлы
ps aux | grep "python"   # Найти python процессы
cat file.txt | wc -l     # Посчитать строки

# Примеры комбинаций
history | grep "git"     # Найти команды git в истории
cat *.log | sort | uniq -c  # Подсчитать уникальные строки в логах
```

## 15. Шпаргалка

| Команда | Что делает |
|---------|------------|
| `ls -la` | Посмотреть содержимое папки (включая скрытые файлы) |
| `cd` | Перемещаться между папками |
| `pwd` | Узнать текущую папку |
| `cp` | Копировать файлы и папки |
| `mv` | Переместить или переименовать |
| `rm` | Удалить файлы |
| `rm -rf` | Удалить папку с содержимым (осторожно!) |
| `mkdir` | Создать новую папку |
| `touch` | Создать пустой файл |
| `cat` | Показать содержимое файла |
| `grep` | Поиск текста в файлах |
| `chmod` | Изменить права доступа |
| `chown` | Изменить владельца файла |
| `sudo` | Выполнить команду от имени root |
| `man` | Показать справочную информацию |
| `--help` | Краткая справка по команде |



## 16. Пример для практики

```bash
# Создать структуру проекта
mkdir -p ~/my_project/{src,logs,config}
cd ~/my_project

# Создать файлы
echo "#!/bin/bash" > src/start.sh
echo "echo 'Starting...'" >> src/start.sh

# Дать права на исполнение
chmod +x src/start.sh

# Поиск всех .sh файлов
find . -name "*.sh"

# Подсчитать количество строк во всех .sh файлах
cat src/*.sh | wc -l

# Архивировать проект
tar -czf my_project.tar.gz ~/my_project
```
